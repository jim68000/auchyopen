Auchy Open Source Auction

Hello, world!

The aim of this project is to create an open source auction style website freely available to organizations that wish to setup online auctions for charity and also an an exercise in online auction software to allow developers and engineers to find out moreb and contribute to technology in this area.

Freecycle Meets Ebay

Things are only just starting, I'm currently in the early phases of requirements gather, defining the service, figuring out the architecture and working out development methods.

Oh and this is supposed to be the 'in' color for 2014 according to Pantone... I'm not convinced.

Mission

Provide an open source platform for local person to person auctions to with the proceeds going to charity.

The requirements for the service are currently being documented. Click here to read more

Project

An open source project that brings together a number of 'modules' to provide an open, flexible and reliable technology stack to host online auction sites.
